## Description

An event-driven multi-seller marketplace backend.
Built with Nestjs.

- 🛍️ Order management.
- ％ Product discounts.
- 🎟️ Voucher management.
- 🤑 Customer refund wallet.
- 🚚 Shipping rates.
- 🧾 Tax configuration per region.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
